Proyecto


¿Puedo ser gerente de varios proyectos?
Usted puede ser gerente de todos los proyectos que usted cree.

¿Cuántos usuarios puedo agregar a mi proyecto?
Usted puede agregar la cantidad de usuarios que desee.

¿Puedo darme de baja de un proyecto del que no soy gerente?
No. Sólo el gerente de proyecto te puede dar de baja (sacarte de proyecto).

¿Puedo crear nuevos permisos para agregarlos a mi proyecto?
No le está permitido a ningún usuario del sistema la creación de nuevos permisos.

Fase


¿Cuántas fases pueden tener mis proyectos?
Sus proyectos pueden tener la cantidad de fases que desee.

¿Se pueden eliminar fases una vez iniciado un proyecto?
Una vez iniciado el proyecto ya no es posible eliminar fases.

¿Cuándo puedo aprobar mis fases?
Una vez que todos los ítems y las líneas bases estén aprobadas.

¿Puedo agregar usuarios por fases?
Sí. Puede agregar como máximo una cantidad igual a la de usuarios que forman parte del proyecto.

Ítem


¿Cuántos ítems pueden tener mis fases?
Las fases pueden tener cantidad que usted necesite de ítems.

¿Qué necesito para aprobar mis ítems?
Necesita que esten relacionados con otros ítems de la misma fase u otra fase.

¿Puedo usar el mismo ítem en dos líneas bases?
No. Un ítem sólo puede formar parte de una línea base.

¿Puedo visualizar el impacto de un ítem no aprobado?
No. Es necesario aprobar el ítem para efectuar el cálculo.

¿Para que sirve el versionado de ítems?
Le sirve para deshacer los cambios de un ítem y regresar a la última versión guardada.

¿El versionado afecta a todo el proyecto?
Sí. Afecta a todas las relaciones directas e indirectas del ítem y por consecuencia, a todo el proyecto.

Línea Base


¿Cuántas líneas bases pueden tener mis fases?
Las fases pueden tener cantidad que usted necesite de líneas bases.

¿Qué significa "Romper" una línea base?
Implica que todos los elementos de esa línea base son liberados para usarlos y así trabajar nuevamente con ellos.

¿Cuántos elementos pueden tener las líneas bases?
Las necesarias que usted requiera para formarlas.

¿Qué necesito para aprobar una línea base?
Sólo necesita que los elementos elegidos para formar parte de la línea base estén aprobados.

Tipo de Ítem


¿Cuántos tipos de ítems puedo tener en mi proyecto?
Sólo se permite un tipo de ítem por fase.

¿Puedo modificar los tipos de ítems que importo?
No. Sólo se permite la edición de los tipos de ítems creados.

¿Cuándo debo importar un tipo de ítem?
Siempre que el proyecto no haya iniciado.

¿Cuándo debo crear el un tipo de ítem?
En cualquier momento antes de iniciar el proyecto donde desea usarlo.
